# ☁️ Files on Cloud

A **cloud-based file sharing and management web app** that allows users to **upload, store, and share files online** securely.  
Each uploaded file generates a **unique share code**, enabling other users to **access and download the file easily** using that code.

---

## 🚀 Features

- 📤 **Upload Files Easily** – Users can upload files directly from their device.
- 🔐 **Unique Share Code System** – Each uploaded file gets a unique code that can be shared securely.
- 📥 **Access Shared Files** – Anyone with the share code can download the corresponding file.
- 🗂️ **Organized File Management** – Clean and minimal interface to keep file sharing simple.
- 🌐 **Deployed on Render** – Fully functional and accessible via a public URL.
- ⚙️ **Future-Ready Design** – Can be easily integrated with cloud storage services like AWS S3 or Google Cloud Storage. More features like shareble link and qr code can be implemented.

## ⚙️ How It Works

1. **Upload a file** using the upload form.  
2. The server generates a **unique share code** for that file.  
3. Share the code with anyone you want.  
4. The recipient enters the **share code** in the access form to **download the file**.  

Simple. Secure. Fast. ⚡
